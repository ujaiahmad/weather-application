# Weather Application
Design inspired by: https://dribbble.com/shots/20675054-Mobile-Weather-app 

Features:
1. Location of device (With permission)
2. Timestamp
3. Description of weather
4. Temperature
5. Summary
6. Wind, Humidity and visibility
7. Daily forecast
